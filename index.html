La clase String incluye métodos y propiedades para procesar string. Un string es un array de caracteres,con un índice entre 0 y número_de_caracteres -1. Cada índice representa un carácter del string. Método length devuelve la longitud, número de caracteres contenidos en el string : 'ciudad'.length => 6 Utilizando el string como un array:'ciudad'[2] => ‘u’ . Devuelve el carácter del indice 2(comienza en 0).
